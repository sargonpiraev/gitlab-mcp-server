# GitLab API MCP Server

MCP server for gitlab integration

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Set up environment:
```bash
cp .env.example .env
# Edit .env with your API credentials
```

3. Start the server:
```bash
npm start
```

## Configuration

Add to your MCP client config:

```json
{
  "mcpServers": {
    "gitlab-mcp-server-mcp-server": {
      "command": "node",
      "args": ["~/path/to/gitlab-mcp-server/dist/index.js"],
      "env": {
        "GITLAB_API_KEY": "your_gitlab_api_key"
      }
    }
  }
}
```
